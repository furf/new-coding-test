getPoissonDistribution(lambda) → {Array.<number>}

Generate the distribution of Poisson probabilities for the most likely number of events. The range is calculated by iterating the Poisson probabilities from x = 0 to x = maximum non-zero probability. The maximum non-zero probability will occur to the right of the mean (x > lambda) when f crosses the x-axis (f <= 0). These values will be used to create a Poisson event randomizer.

getRandomizer(values) → {function}

Generate a random integer generator from 0–n where n is the length of the array of values.* The array should contain numerical thresholds from 0–1. For example, given an array of values [0.1, 0.4, 0.9], you would have a: * 10% chance of 0 (random number is between 0 and 0.1) * 30% chance of 1 (random number is between 0.1 and 0.4) * 50% chance of 2 (random number is between 0.4 and 0.9) * 10% chance of 3 (random number is between 0.9 and 1) *This function should be changed to normalize the values using the min and max of the range to represent 0 and 1. This function currently has the tiny flaw of returning n + 1 if the random number is greater than the highest value in the array, but for the purposes of this assignment it is fine. The Poisson distribution function returns max values that are 1 or are relatively indescernible.

values Array.<number> Array of numerical values from 0–1
